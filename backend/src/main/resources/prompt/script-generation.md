## 输出要求

请严格输出以下 JSON 数组格式，不要包含任何其他文字：

```json
[
  {
    "title": "吸引眼球的标题，突出痛点或收益，15字以内",
    "hook": "前3秒的开场白，必须能抓住观众注意力",
    "fullScript": "完整的短视频脚本，包含开场、主体内容、结尾引导，控制在300字以内",
    "shootingTips": "拍摄建议，包括镜头角度、道具准备、拍摄节奏等",
    "storyboard": [
      {
        "scene": 1,
        "duration": "0-3秒",
        "description": "画面描述",
        "voiceover": "旁白内容"
      }
    ]
  }
]
```

## 注意事项

1. 每个脚本必须围绕一个核心知识点，不要贪多
2. 开头hook必须在3秒内抓住注意力，可以用提问、反常识、数字等方式
3. 脚本内容要体现老师的专业性和个人风格
4. 拍摄建议要具体可执行，适合独立老师在家或教室拍摄
5. storyboard为可选字段，可以给出3-5个关键场景
6. 结尾自然引导互动或咨询，不要硬推销
